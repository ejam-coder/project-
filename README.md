# IA Game Project – Tópicos de IA

Este proyecto consiste en el desarrollo de un mini-juego interactivo que utiliza visión por computadora y técnicas de IA para detectar poses, gestos o elementos visuales que controlan la mecánica del juego.

## Características clave del proyecto
- Detección de poses / manos / rostro (según mecánica final)
- Procesamiento de imágenes en tiempo real
- Control del juego mediante IA
- Juego ejecutable en web o móvil
- Documentación en formato IEEE (Overleaf)

## Estructura inicial
- `src/` – código fuente (frontend, backend, modelos)
- `datasets/` – datos crudos y procesados
- `docs/` – especificaciones y documentación
